Use deepwiki tools to get wiki structure, read wiki contents, or ask questions about GitHub repositories.

Available tools:
- `deepwiki_read_wiki_structure` - Get documentation topics for a repo
- `deepwiki_read_wiki_contents` - View documentation about a repo
- `deepwiki_ask_question` - Ask any question about a repo (AI-powered, context-grounded)
